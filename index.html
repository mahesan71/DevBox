It's wherever you are.

DevBox is available for you on multiple platforms:

  • As an offline desktop application on macOs, Windows, Linux.

  • As a browser extension in Chrome or Firefox.

  • As a web application in any modern browser.

It's offline.

DevBox is offline first, meaning anything that happens on your computer stays on your computer, in addition:

  • You are safe, there is no tracking of any kind.

  • No dark patterns, no hidden features.

It's old school.

After a one time purchase DevBox (desktop application) can be used forever, no DRM, no license, free updates for life!

It's modern.

A nice, clean, functional UI which looks the same on every platform with a dark and light theme.

General

What technologies are used?

DevBox is basically a web application which is using the operating systems webview. It's built using two technologies: Tauri - creates desktop applications from basically web pages Mint - programming language to create single page applications (or in this case desktop :)) Mint UI - a user interface library (in Mint) to write applications in Mint
